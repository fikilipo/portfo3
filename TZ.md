# «Панельный Код» / The Panelka Grid

Интерактивный манифест CSS-дизайна в форме фасада позднесоветской панельки.

## Концепция

Сайт — это один большой дом. Фасад собран из CSS Grid. В каждом окне — отдельный визуальный этюд: бегущая строка, шторы, глитч, аквариум, телевизор, кот, гирлянда. Никаких фотографий, иконок и SVG-картинок: только HTML, CSS и немного JS-логики на переключение состояний.

Главная эмоция — уютная меланхолия. Ритм спальных районов, серость бетона, тёплый свет в окнах под вечер. Сайт должен ощущаться сделанным человеком, который любит и панельки, и веб.

## Структура страницы

1. **Небо.** Фон во всю высоту первого экрана. Меняется по времени суток (день / сумерки / ночь) через CSS-переменные.
2. **Крыша.** Антенны, спутниковая тарелка, труба с лёгким «дымом». Сделано на `::before` / `::after` и `box-shadow`.
3. **Фасад.** Основной CSS Grid: 6 этажей × 5 окон + балконы. Используется `grid-template-areas`, чтобы расставить нестандартные окна (балкон шире окна, лестничная клетка вертикально на два этажа и т. п.). Между этажами — межпанельные швы (бетонные пояса).
4. **Цоколь.** Подъезд с дверью, скамейка, мусорный контейнер, фонарь. Можно одной полосой внизу.
5. **Манифест.** Текстовый блок под домом: название проекта, лид, краткие тезисы о CSS-сетке как метафоре города.
6. **Подвал.** Год, автор, технологии, ссылка на исходник.

## Технические требования

- **Стек:** Vite + React 18 + TypeScript, Tailwind CSS как база, Framer Motion для отдельных переходов. Основная визуальная логика — на чистом CSS.
- **Сетка:** хотя бы один уровень верстки использует `grid-template-areas`. Дом адаптивен: на мобиле — 3 окна в ряд, на десктопе — 5–6.
- **Время суток:** переключатель в углу (день / вечер / ночь). Реализован через корневые CSS-переменные (`--sky-top`, `--wall-base`, `--window-glow-*`). Анимируется плавно (`transition` на переменных или на конкретных свойствах).
- **Текстуры:** бетон, штукатурка, рамы, отливы — собраны из `repeating-linear-gradient`, `radial-gradient`, `box-shadow`, `filter` и псевдоэлементов. Никаких растровых картинок.
- **Контент:** весь текст на русском языке. Тон — спокойный, без канцеляритов и без типичных оборотов нейросети («давайте погрузимся», «в этой статье мы», «важно отметить» и т. п.).
- **Код:** без декоративных и поясняющих комментариев. Имена компонентов и классов говорят сами за себя.

## Список окон (минимум 16 уникальных)

Каждое окно живёт самостоятельно. Часть — статичные миниатюры, часть — анимированные, часть — интерактивные (реагируют на наведение / клик).

1. **Бегущая строка.** Кинетическая типографика, текст про город, бесконечная прокрутка на CSS keyframes.
2. **Шторы.** `clip-path` раздвигается при наведении, открывая силуэт за окном.
3. **Глитч «РУССКИЙ ДИЗАЙН».** Кириллический заголовок с RGB-сдвигом и дрожью.
4. **Телевизор.** Шум из conic/repeating gradient, поверх — синее свечение и логотип канала, который морфит.
5. **Кот на подоконнике.** Силуэт из `border-radius` и `box-shadow`, моргает раз в несколько секунд.
6. **Дождь.** Капли — псевдоэлементы, падают по линейной анимации, оседают на стекле.
7. **Растение.** Горшок и листья из CSS-фигур, листья слегка покачиваются.
8. **Дискошар.** `conic-gradient` + `mix-blend-mode`, медленно вращается, бросает блики.
9. **Лампа Ильича.** Голая лампочка на проводе, качается, тёплое свечение через `filter: blur` + `box-shadow`.
10. **Гирлянда.** Новогодние лампочки разных цветов, пульсируют не в такт.
11. **Аквариум.** Сине-зелёный градиент, рыбки — точки, дрейфуют по `@keyframes`.
12. **Заколоченное окно.** Доски крест-накрест, трещины на стекле.
13. **Сигаретный дым.** Тлеющий огонёк, дым поднимается и растворяется.
14. **Ковёр на стене.** Геометрический орнамент из градиентов, виден через окно.
15. **Старый монитор.** CRT-развёртка, бегущие строки, мерцание.
16. **Тёмное окно.** Просто чёрное, иногда мелькает отражение.
17. **Балкон с бельём.** Верёвка с одеждой, лёгкое колыхание.
18. **Подъезд.** Лестничная клетка на два этажа: лампа дневного света мерцает.

Часть окон можно свободно заменить или добавить, главное — чтобы фасад не выглядел повторяющимся.

## Интеракции

- Переключатель времени суток (день / вечер / ночь).
- Наведение на окно — подсветка рамы, более активная анимация контента.
- Клик по окну — окно «открывается» (лёгкая 3D-инверсия через `transform`), фоновый шум приглушается.
- Клавиша `N` — мгновенно ночь, `D` — день. Лёгкий пасхальный жест.

## Критерии приёмки

- Сайт собирается без ошибок (`npm run build`), линт чистый.
- На десктопе фасад читается как один цельный дом, не разваливается.
- На мобильном (≤ 480px) сетка сжимается, окна остаются интерактивными.
- В ночном режиме светятся минимум 8 окон, причём цвета разные.
- В коде нет сторонних картинок и SVG-иллюстраций — только текст, разметка и стили.
- Текст не выглядит сгенерированным: короткие фразы, конкретика, без лишних оборотов.
